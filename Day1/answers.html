The Differences Between let, const and var

A var variable can be redeclared and updated.
A let variable be be updated but not redeclared.

        let a=20;
        let a=10;  // error as let variable cannot be redeclared
        a=30;     // allowed changes value to 30
        {
            let a=25;
            console.log(a);  // prints 25
        }
        console.log(a);  // prints 30

        let is blockscoped
        

        var a=20;
        var a='hello'; // allowed as var can be redeclared
        {
            a="world";  //changes value to world
        }
        console.log(a);  // prints world

        Var is globally scoped
    

        Const a=20;
        a=10; //error const cannot be changed
        const a=30; // Uncaught SyntaxError: Identifier 'a' has already been declared

        const b=[1,2,3]
        b.push(4)    // b = [1, 2, 3, 4]
        b=[2]   //Uncaught TypeError: Assignment to constant variable.

        Const is also globally scoped
